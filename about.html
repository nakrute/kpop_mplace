About K-Card Market

K-Card Market is a demo marketplace concept for buying/selling K-pop photocards with a TCGplayer-style browsing experience. This site currently uses localStorage for wishlist, cart, requests, and demo orders.

What’s working now

Browse + filters + sorting, wishlist, cart drawer, checkout demo, and a seller dashboard demo.

What’s next

Supabase Auth, catalog + listings tables, seller profiles, real images, and real checkout via Stripe.
